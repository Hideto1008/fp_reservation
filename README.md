# フィナンシャルプランナー予約システム

このアプリは、ユーザーがフィナンシャルプランナー（FP）との相談時間を予約できるシステムです。
FPは予約枠を設定し、ユーザーはその空き枠を選択して予約を行います。

---

## 特徴

- **予約単位**: 1枠30分
- **予約可能時間**:
  - **平日**: 10:00 ~ 18:00
  - **土曜日**: 11:00 ~ 15:00
  - **日曜日**: 休業
- **認証機能**:
  - ユーザーとFPのメールアドレス・パスワードによる登録・認証

---

## セットアップ手順

以下の手順でアプリケーションをセットアップしてください。

### 1. リポジトリをクローン
以下のコマンドでリポジトリをクローンします
```bash
git@github.com:Hideto1008/fp_reservation.git
```
クローンしたらプロジェクトアプリのディレクトリに移動します
```bash
cd fp_app
```

### 2. Dockerのビルドとコンテナの起動

以下のコマンドを実行して、アプリケーションをビルド・起動します。

```bash
docker compose up --build -d
```
### 3. Docker環境でのコマンド操作
以下のコマンドを実行して、Dockerの環境でコマンドを実行できるようにします。

```bash
docker exec -it fp_app /bin/bash
```
### 4. データベースの作成

以下のコマンドを実行して、データベースを作成します。

```bash
rails db:create
```

### 5. データベースのマイグレーション

以下のコマンドを実行して、データベースのマイグレーションを行います。

```bash
make
```

###6. 非同期処理の更新
以下のコマンドを実行してwhenever gemを使って非同期処理の設定を更新します。

```bash
bundle exec whenever --update-crontab
```

更新が完了したら、以下のコマンドを実行して非同期処理の開始を指示します。



### 6. アプリの開始
以下のコマンを実行してrailsアプリを立ち上げます

```bash
rails s
```
http://localhost:3000/
にアクセスして以下の画面が表示されれば、セットアップ完了です！


### 7. コンテナを終了

以下のコマンドを実行して、`bash` シェルを終了し、コンテナから抜けます。

```bash
exit
```
コンテナから抜けたら以下のコマンドを実行してコンテナを終了させてください

```bash
docker compose down
```
